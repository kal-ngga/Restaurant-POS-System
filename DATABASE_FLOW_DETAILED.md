## ðŸ“Œ KEY POINTS

1. **Data User Tersimpan**: Saat register, data langsung di-INSERT ke tabel `users`
2. **Password di-Hash**: Password tidak disimpan plain text, tapi di-encrypt dengan bcrypt
3. **Email Unique**: Tidak boleh ada 2 user dengan email sama
4. **Session Auto-Created**: Setelah register, user otomatis di-login
5. **Login = Database Query**: Setiap login, Laravel query database untuk cek email & password
